# Welcome to Slidev

Building presentations with Markdown.

---

# Why Slidev?

- 📝 **Markdown-based** - use your favorite editor
- 🎨 **Web-powered** - full control over everything
- 🚀 **Fast** - hot reloading and optimized builds
- 📦 **NPM-compatible** - use any Vue components

# How to add more?

Just create another `.md` file in this folder and run:

```bash
npm run dev -- your-file.md
```
